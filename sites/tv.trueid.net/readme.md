# tv.trueid.net

https://tv.trueid.net/

### Download the guide

```sh
npm run grab -- --site=tv.trueid.net
```

### Update channel list

```sh
npm run channels:parse -- --config=./sites/tv.trueid.net/tv.trueid.net.config.js --output=./sites/tv.trueid.net/tv.trueid.net.channels.xml
```

### Test

```sh
npm test -- tv.trueid.net
```
