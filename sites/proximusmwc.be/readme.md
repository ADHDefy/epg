# proximusmwc.be

http://proximusmwc.be/

### Download the guide

```sh
npm run grab -- --site=proximusmwc.be
```

### Update channel list

```sh
npm run channels:parse -- --config=./sites/proximusmwc.be/proximusmwc.be.config.js --output=./sites/proximusmwc.be/proximusmwc.be.channels.xml
```

### Test

```sh
npm test -- proximusmwc.be
```
